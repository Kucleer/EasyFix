# 项目名称

EasyFix

## 项目概述

这是一个用于 [学生错题整理] 的轻量级应用。项目的主要功能包括 [错题上传]、[错题统计] 和 [辅助学习]。

## 开发者背景

- **技能水平**：初学者，正在学习编程
- **主要语言**：Python / JavaScript / [其他]
- **学习目标**：掌握基础编程技能，能够独立完成小型项目
- **偏好**：喜欢简洁易懂的代码，注释要详细,无特殊要求交流均为中文

## 技术栈

- **后端**：Python
- **前端**：Vue
- **数据库**：MySQL

## 代码风格

### 命名规范

- **变量命名**：使用小写下划线命名法（snake_case）
- **函数命名**：使用动词或动词短语
- **类命名**：使用大写开头命名法（PascalCase）
- **常量命名**：全大写下划线分隔
  

### 注释要求

- 每个函数前添加文档字符串，说明功能
- 复杂逻辑添加行内注释说明
- 使用中文注释，便于理解
