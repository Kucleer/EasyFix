# 上传错题页面重构设计

## 概述

重构上传错题页面，移除OCR功能，改为直接编辑全部字段的可选图片上传卡片式表单布局。

## 设计方案

### 布局结构

```
┌─────────────────────────────────────────────┐
│  新增错题                                    │
├─────────────────────────────────────────────┤
│  ┌─────────────────────────────────────┐    │
│  │ 📷 图片上传卡片（蓝色标题栏）         │    │
│  │ - 拖拽/点击上传，可上传多张           │    │
│  │ - 已上传图片横向排列，可预览/删除       │    │
│  └─────────────────────────────────────┘    │
│  ┌─────────────────────────────────────┐    │
│  │ 📝 基本信息卡片（蓝色标题栏）          │    │
│  │ - 学科（必填）                        │    │
│  │ - 年级 / 学期                         │    │
│  └─────────────────────────────────────┘    │
│  ┌─────────────────────────────────────┐    │
│  │ 📖 题目内容卡片（绿色标题栏）          │    │
│  │ - 题目（textarea）                    │    │
│  │ - 答案（textarea）                    │    │
│  │ - 解析（textarea）                    │    │
│  └─────────────────────────────────────┘    │
│  ┌─────────────────────────────────────┐    │
│  │ ⭐ 难度与分类卡片（橙色标题栏）        │    │
│  │ - 难度星级选择                        │    │
│  │ - 错误类型（多选，与知识点联动）       │    │
│  │ - 知识点（可搜索/创建）               │    │
│  │ - 标签（多选）                        │    │
│  └─────────────────────────────────────┘    │
│  ┌─────────────────────────────────────┐    │
│  │ [保存错题按钮]                        │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

### 卡片配色方案

| 卡片 | 标题栏背景色 | 标题文字色 |
|------|-------------|-----------|
| 图片上传 | `#409eff` (蓝色) | 白色 |
| 基本信息 | `#409eff` (蓝色) | 白色 |
| 题目内容 | `#67c23a` (绿色) | 白色 |
| 难度与分类 | `#e6a23c` (橙色) | 白色 |

### 卡片样式

- 圆角：`border-radius: 12px`
- 阴影：`box-shadow: 0 4px 16px rgba(0,0,0,0.08)`
- 内边距：标题栏 12px 16px，内容区 16px
- 卡片间距：16px
- 标题栏使用渐变色

### 连续录入功能

连续录入时，自动保留上次选择的基本信息：

| 字段 | 连续录入保留 |
|------|------------|
| 学科 | ✅ 保留（localStorage） |
| 年级 | ✅ 保留（localStorage） |
| 学期 | ✅ 保留（localStorage） |
| 题目/答案/解析 | ❌ 每次清空 |
| 图片 | ❌ 每次清空 |
| 难度/错误类型/知识点/标签 | ❌ 每次清空 |

### 移除功能

- 移除 el-steps 向导流程
- 移除 OCR 识别相关所有功能（上传、识别、结果展示）
- 移除多步骤切换逻辑
- 移除手动录入模式

### 保留功能

- 图片上传功能（改为可选）
- 知识点与错误类型联动
- 标签多选
- localStorage 持久化

## 实现文件

- `frontend/src/views/Upload.vue` — 完全重写

## 验收标准

1. 无需OCR，图片上传为可选项
2. 单页完成所有字段编辑
3. 卡片分区式布局，与详情页风格一致
4. 知识点/错误类型仍保持联动
5. 连续录入时保留学科/年级/学期
6. 表单提交后清空题目内容但保留基本信息
