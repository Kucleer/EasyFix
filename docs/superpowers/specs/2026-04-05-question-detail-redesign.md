# 错题详情页 UI 美化设计

## 概述

美化错题详情弹窗的视觉设计，采用卡片分区式布局，提升信息层次感和可读性。

## 设计方案

### 布局结构

```
┌─────────────────────────────────────────────────────────┐
│  原图展示区（横向滚动图册，支持大图预览）                   │
├───────────────────────┬─────────────────────────────────┤
│  题目卡片（蓝色标题栏） │  答案卡片（绿色标题栏）           │
├───────────────────────┴─────────────────────────────────┤
│  解析卡片（橙色标题栏）                                   │
├─────────────────────────────────────────────────────────┤
│  元数据卡片（灰色标题栏）                                 │
│  [难度★★★★☆] [年级/学期] [错误类型] [知识点]            │
├─────────────────────────────────────────────────────────┤
│  标签区域（彩色胶囊标签云）                               │
├─────────────────────────────────────────────────────────┤
│  相似题卡片（紫色标题栏，可展开）                         │
└─────────────────────────────────────────────────────────┘
```

### 卡片配色方案

| 卡片 | 标题栏背景色 | 标题文字色 |
|------|-------------|-----------|
| 题目 | `#409eff` (蓝色) | 白色 |
| 答案 | `#67c23a` (绿色) | 白色 |
| 解析 | `#e6a23c` (橙色) | 白色 |
| 元数据 | `#606266` (深灰) | 白色 |
| 相似题 | `#9c27b0` (紫色) | 白色 |

### 难度展示

- 样式：大号黄色星星 `★★★★★`
- 难度4显示：`★★★★☆`（实心黄星 + 空心星）
- 字体大小：18px，字母间距 3px
- 颜色：统一黄色 `#f5c518`

### 卡片样式

- 圆角：`border-radius: 12px`
- 阴影：`box-shadow: 0 4px 16px rgba(0,0,0,0.08)`
- 内边距：标题栏 12px 16px，内容区 16px
- 卡片间距：16px

### 原图展示区

- 横向滚动，图片宽高 180px x 180px
- 圆角图片：`border-radius: 8px`
- hover 放大效果：`transform: scale(1.05)`
- 支持点击大图预览

### 标签区域

- 使用 Element Plus el-tag 彩色胶囊样式
- 标签间距：8px
- 无卡片包裹，纯标签云展示

## 实现文件

- `frontend/src/views/Questions.vue` — 详情弹窗模板和样式

## 验收标准

1. 详情页显示为卡片分区式布局
2. 难度显示为大号黄色星星，与难度数字对应
3. 题目/答案/解析/元数据/相似题各有独立卡片和对应颜色标题栏
4. 卡片有圆角和阴影，视觉效果有层次感
5. 原图展示支持横向滚动和点击预览
