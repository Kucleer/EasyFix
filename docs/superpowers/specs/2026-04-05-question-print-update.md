# 错题列表打印更新设计

> **Goal:** 更新练习集PDF打印样式，紧凑排版、添加标注信息、难度改用彩色星号显示

## 布局结构（每道题）

```
┌──────────────────────────────────────────────────────┐
│ [ID: 123]  第1题   ★★★☆☆   知识点: 方程   错误类型: 计算 │
│──────────────────────────────────────────────────────│
│ 题目内容文本                                            │
└──────────────────────────────────────────────────────┘
```

**改动点：**
1. 删除底部60px虚线答题区域
2. 增加 ID、知识点(error_type)、错误类型(knowledge_point) 标注
3. 难度从文字改为彩色星号（★填充色，☆灰色）
